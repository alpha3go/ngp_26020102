# ✨ 원형 시간표 프로젝트 블루프린트

## 📜 개요

누구나 쉽고 직관적으로 자신만의 원형 시간표를 만들고 관리할 수 있는 웹 서비스입니다. 이 서비스는 사용자가 시간 관리를 시각적으로 재미있게 할 수 있도록 돕는 것을 목표로 합니다.

## 🎨 디자인 및 기능 명세

### 1. **핵심 기능**
- **원형 시간표 UI**: 24시간을 나타내는 원형 다이얼 위에 시간표 항목을 시각적으로 표시합니다.
- **항목 추가/수정/삭제**: 시간(시작/종료), 활동 이름, 색상을 지정하여 시간표에 항목을 동적으로 추가, 수정, 삭제할 수 있습니다.
- **로컬 저장소**: 사용자가 생성한 시간표 데이터는 브라우저의 로컬 저장소(`localStorage`)에 자동 저장되어, 페이지를 새로고침해도 데이터가 유지됩니다.
- **PNG로 내보내기**: 완성된 시간표를 이미지 파일로 다운로드할 수 있는 기능을 제공합니다.
- **반응형 디자인**: 데스크톱과 모바일 기기 모두에서 최적화된 화면을 보여줍니다.

### 2. **디자인 컨셉**
- **레이아웃**: 화면 중앙에 큰 원형 시간표를 배치하고, 시간표를 제어하는 입력 폼과 버튼들은 측면이나 하단에 배치하여 사용자가 핵심 기능에 집중할 수 있도록 합니다.
- **색상**: 활기차고 긍정적인 느낌을 주는 밝은 색상 팔레트를 기본으로 사용하며, 사용자가 각 항목의 색상을 직접 선택할 수 있는 컬러 피커를 제공합니다.
- **타이포그래피**: 현대적이고 가독성이 좋은 `Pretendard`와 같은 웹 폰트를 사용하여 명확한 정보 전달을 돕습니다.
- **인터랙션**: 항목이 추가되거나 삭제될 때 부드러운 전환(transition) 효과를 적용하고, 버튼과 같은 상호작용 요소에는 미묘한 그림자나 'glow' 효과를 주어 시각적 피드백을 강화합니다.
